# ✨ Glow & Care Marketplace

Bem-vindo ao **Glow & Care**, o seu destino digital para uma rotina de skincare impecável. Este é um marketplace estático focado em elegância, simplicidade e conversão.

## 🚀 Sobre o Projeto

O objetivo deste projeto é fornecer uma vitrine de produtos de skincare com um design moderno e responsivo. Cada detalhe foi pensado para transmitir confiança e sofisticação, desde a escolha das fontes até a suavidade das cores.

## 🎨 Design System

As diretrizes visuais deste projeto estão documentadas em [design-system.md](./design-system.md). Utilizamos uma paleta de cores inspirada em tons naturais e texturas de seda para criar uma experiência relaxante para o usuário.

## 🛠️ Tecnologias Utilizadas

- **HTML5** (Estrutura semântica)
- **CSS3** (Tailwind CSS para agilidade e precisão)
- **Fontes do Google** (Playfair Display & Montserrat)
